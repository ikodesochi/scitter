# 📝 Учимся писать псевдокод

## Что такое псевдокод?

Это описание того, что делает программа, **простыми словами**.

Ты не пишешь код. Ты пишешь **план** — как в сочинении:
- Сначала я сделаю это
- Потом проверю то
- Если случится вот это — сделаю так

## Зачем нужен план?

Представь что ты строишь дом из LEGO:
- Без инструкции — получится куча кирпичей
- С инструкцией — получится замок

Псевдокод = инструкция для твоего кода.

## Простой пример

### Задача: мышь идёт вправо

**Плохо (слишком сложно):**
```
Реализовать перемещение объекта Mouse по оси X с инкрементом дельты
```

**Хорошо (простыми словами):**
```
Когда нажата стрелка вправо:
  Двигаем мышь вправо на 4 пикселя
  Если там стена — не двигаем
```

---

## Правила хорошего псевдокода

1. Пиши **короткими фразами**
2. Используй слова: **ЕСЛИ, ТО, ИНАЧЕ, ПОКА, ДЛЯ КАЖДОГО**
3. Не пиши код — пиши **смысл**
4. Один шаг = одна строка

## Слова-помощники

| Слово | Значение | Пример |
|-------|----------|--------|
| ЕСЛИ | Проверка условия | ЕСЛИ нажата W |
| ТО | Что делаем | ТО двигаем вверх |
| ИНАЧЕ | Если условие не сработало | ИНАЧЕ стоим на месте |
| ПОКА | Повторяем пока условие верно | ПОКА не дошли до стены |
| ДЛЯ КАЖДОГО | Перебираем список | ДЛЯ КАЖДОГО сыра на карте |
| ЖДАТЬ | Пауза | ЖДАТЬ 2 секунды |

## Примеры из нашей игры

### Движение мыши (Мурад)
```
ФУНКЦИЯ движение мыши:
  ЕСЛИ нажата W:
    двигаем мышь вверх
  ЕСЛИ нажата S:
    двигаем мышь вниз
  ЕСЛИ нажата A:
    двигаем мышь влево
  ЕСЛИ нажата D:
    двигаем мышь вправо

  ЕСЛИ впереди стена:
    не двигаем
    прижимаем к стене
```

### Кот преследует мышь (Мурад)
```
ФУНКЦИЯ поведение кота:
  ЕСЛИ кот видит мышь:
    бежим к мыши
    показываем "!"
  ИНАЧЕ ЕСЛИ кот искал мышь но потерял:
    идём к месту где видели мышь последний раз
    ЕСЛИ не нашли за 2 секунды:
      возвращаемся к патрулированию
  ИНАЧЕ:
    ходим случайно по своей территории
    иногда останавливаемся
```

### Сбор сыра (Миша)
```
ФУНКЦИЯ сбор сыра:
  ДЛЯ КАЖДОГО сыра на карте:
    ЕСЛИ мышь коснулась сыра И сыр ещё не собран:
      сыр исчезает
      счётчик сыра + 1
      показываем "+🧀" на 3 секунды
  
  ЕСЛИ собрали весь сыр:
    активируем лунку
    показываем "!!!"
```

### Масляное пятно (Денис)
```
ФУНКЦИЯ масляное пятно:
  ЕСЛИ мышь наступила на масло:
    замедляем мышь (скорость делим на 3)
    показываем что мышь скользит
    ЖДАТЬ 2 секунды
    возвращаем обычную скорость
    пятно исчезает
```

### Пасхалка (Тимур)
```
ФУНКЦИЯ пасхалка:
  ЕСЛИ игрок нажал I, потом C, потом A, потом T подряд:
    меняем цвет кота на красный
    показываем "Пасхалка найдена!"
    ЖДАТЬ 5 секунд
    возвращаем обычный цвет кота
```

### Таблица рекордов (Миша)
```
ФУНКЦИЯ сохранить рекорд:
  ЕСЛИ текущий счёт больше рекорда:
    сохраняем новый рекорд
    показываем "Новый рекорд!"
  
  При загрузке игры:
    читаем рекорд из памяти
    показываем его на экране
```

## Твоя очередь!

1. Открой свой файл `pseudocode.md` в ветке
2. Напиши псевдокод для своей задачи **простыми словами**
3. Используй слова-помощники: ЕСЛИ, ТО, ИНАЧЕ, ПОКА
4. Покажи наставнику
5. После одобрения — пиши код!
