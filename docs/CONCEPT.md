# 🧀 Концепция игры Scitter

## 🐭 О чём игра

Scitter — минималистичная игра в чёрно-белом стиле. Мышь собирает сыр в лабиринтах, избегая кота. Собака защищает мышь если её покормить. Между уровнями — вентиляция с тараканом-торговцем.

## 🎮 Жанр

Top-down survival maze (лабиринт с видом сверху, выживание)

## 🌐 Платформа

Браузер (HTML, CSS, JavaScript). Не требует установки.

## 🎨 Визуальный стиль

- Чёрно-белая палитра с акцентами (жёлтый сыр, белые кости)
- Персонажи — прямоугольники с ASCII-смайликами (`o_o`, `-_-`, `@_@`)
- Меню — ASCII-графика
- Минимализм во всём

## ⚙️ Главные механики

1. 🧀 Сбор сыра — основная цель уровня
2. 🐱 Избегание кота — кот патрулирует и преследует
3. 🐕 Кормление собаки — кости повышают сытость, собака защищает
4. 🕳️ Лунка — активируется после сбора всех сыров, переход на следующий уровень
5. 🔧 Вентиляция — хаб с тараканом-торговцем между уровнями
6. ⬆️ Улучшения — покупка за сыр (скорость, защита, зрение, костюм)

## 📈 Прогрессия

5 уровней сложности:
1. 🏠 Подземелье — обучение, без собаки
2. 🏙️ Улица — открытое пространство, без собаки
3. 🐕 Дружба — нужно кормить собаку
4. 🛢️ Ловушки — узкие коридоры с препятствиями
5. 👑 Финал — темнота и босс-кот

## 👥 Целевая аудитория

- Дети 10-14 лет (учебный проект по программированию)
- Любители минималистичных игр

## 💻 Технологии

- JavaScript ES6 модули
- Canvas 2D для отрисовки
- Без фреймворков и библиотек
- Git и GitHub для командной работы

## 📐 Принципы разработки

- Модульность: статика отдельно, динамика отдельно
- Всё генерируется случайно, но с гарантией проходимости
- Код комментируется для обучения
- Минимум зависимостей между файлами
