# 📋 TODO.md — План проекта Skitter

## Игровой цикл
- Полный цикл: Уровень 1 → Вентиляция → Уровень 2 → Вентиляция → ... → Уровень 5 → Победа
- Сохранение прогресса между уровнями (сыр, улучшения)
- Экран победы после финального уровня

## Уровни
- 5 уровней с разными механиками (лабиринт, улица, дружба, ловушки, босс)
- Вентиляция с тараканом-торговцем между уровнями
- Случайная генерация карты с проверкой проходимости

## Персонажи
- Мышь: управление, страх, здоровье, улучшения
- Кот: патрулирование, преследование, потеря из виду
- Собака: случайное блуждание, защита от кота, кормление
- Таракан: магазин улучшений за сыр

## Предметы
- Сыр: сбор, активация лунки
- Кости: сбор, кормление собаки
- Лунка: переход между уровнями

## Интерфейс
- Чёрно-белый минимализм
- ASCII-индикаторы над персонажами
- Меню паузы (ESC) со статистикой
- Магазин таракана (E) с улучшениями
- Админ-панель (\\) для отладки

## Ассеты
- Спрайты персонажей
- Текстуры тайлов
- Звуки событий
- Фоны уровней

## Код
- Модульная структура (статика отдельно, динамика отдельно)
- Комментарии для обучения
- Оптимизация и разгрузка файлов
