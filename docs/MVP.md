# 🎯 MVP — Минимальная рабочая версия

MVP (Minimum Viable Product) — это самая простая версия игры, которая уже работает. Всё остальное добавляется потом.

## ✅ Что входит в MVP

### 🐭 Игровой цикл
- [x] Мышь управляется стрелками
- [x] Кот преследует мышь
- [x] Можно собирать сыр
- [x] Лунка активируется после сбора сыра
- [x] Переход между уровнями через лунку
- [x] Вентиляция с тараканом между уровнями

### 🗺️ Уровни
- [x] 5 уровней с разными картами
- [x] Случайная генерация стен и предметов
- [x] Гарантированная проходимость

### 🎒 Предметы
- [x] Сыр (цель уровня)
- [x] Кости (для собаки)
- [x] Лунка (выход)

### 🐕 Персонажи
- [x] Мышь с индикаторами статуса
- [x] Кот с патрулированием
- [x] Собака на уровнях 3 и 5
- [x] Таракан-торговец в вентиляции

### 🖥️ Интерфейс
- [x] Меню паузы по ESC
- [x] ASCII-статистика
- [x] Индикаторы над персонажами

## 🔮 Что будет потом (после MVP)

- [ ] Звуки событий
- [ ] Экран победы
- [ ] Таблица рекордов
- [ ] Больше улучшений у таракана
- [ ] Секретные пасхалки
- [ ] Анимации персонажей
