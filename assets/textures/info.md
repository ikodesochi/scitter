# Текстуры

## Путь
`src/assets/textures/`

## Формат файлов
- Размер: любой
- Формат: PNG

## Список текстур
- [ ] `src/assets/textures/shadow.png` — Тень
- [ ] `src/assets/textures/light.png` — Свет
- [ ] `src/assets/textures/particle.png` — Частицы

## Как добавить
1. Нарисуй или скачай картинку
2. Сохрани в `src/assets/textures/` с правильным именем
3. Поставь `[x]` вместо `[ ]`
