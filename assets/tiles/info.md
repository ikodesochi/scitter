# Тайлы карты

## Путь
`src/assets/tiles/`

## Формат файлов
- Размер: 40×40 пикселей
- Формат: PNG

## Список тайлов
- [ ] `src/assets/tiles/grass.png` — Трава
- [ ] `src/assets/tiles/stone.png` — Камень
- [ ] `src/assets/tiles/wood.png` — Дерево
- [ ] `src/assets/tiles/water.png` — Вода

## Как добавить
1. Нарисуй или скачай картинку
2. Сохрани в `src/assets/tiles/` с правильным именем
3. Поставь `[x]` вместо `[ ]`
