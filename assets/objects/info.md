# Объекты

## Путь
`src/assets/objects/`

## Формат файлов
- Размер: 40×40 пикселей
- Формат: PNG

## Список объектов
- [ ] `src/assets/objects/chest.png` — Сундук
- [ ] `src/assets/objects/crate.png` — Ящик
- [ ] `src/assets/objects/sign.png` — Указатель
- [ ] `src/assets/objects/hole.png` — Норка

## Как добавить
1. Нарисуй или скачай картинку
2. Сохрани в `src/assets/objects/` с правильным именем
3. Поставь `[x]` вместо `[ ]`
