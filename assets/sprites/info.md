# Спрайты персонажей

## Путь
`src/assets/sprites/`

## Формат файлов
- Размер: 32×32 или 64×64 пикселя
- Формат: PNG (с прозрачным фоном)
- Именование: `имя_действие.png`

## Список спрайтов
- [ ] `src/assets/sprites/mouse_idle.png` — Мышь стоит
- [ ] `src/assets/sprites/mouse_run.png` — Мышь бежит
- [ ] `src/assets/sprites/mouse_scared.png` — Мышь напугана
- [ ] `src/assets/sprites/cat_idle.png` — Кот стоит
- [ ] `src/assets/sprites/cat_run.png` — Кот бежит
- [ ] `src/assets/sprites/cat_attack.png` — Кот атакует
- [ ] `src/assets/sprites/dog_idle.png` — Собака стоит
- [ ] `src/assets/sprites/dog_bark.png` — Собака лает

## Как добавить
1. Нарисуй или скачай картинку
2. Сохрани в `src/assets/sprites/` с правильным именем
3. Поставь `[x]` вместо `[ ]`
