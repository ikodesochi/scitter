## 🖥️ КАРТОЧКА ДЕНИСА — УРОВНИ

🎯 **Роль:** Архитектор лабиринтов

📊 **Сложность:** ⭐⭐⭐ (высокая)

📂 **Файлы для работы:**
- `levels/level1.js` — первый уровень
- `levels/level2.js` — второй уровень
- `levels/level3.js` — третий уровень
- `levels/level4.js` — четвёртый уровень
- `levels/level5.js` — пятый уровень

🤝 **Спросить если что:** Иван (спрайты персонажей должны помещаться в твои коридоры), Олег (логика врагов зависит от карты), Мурад (физика проверяет твои стены), Наставник

---

### 📝 ЧТО ТЫ ДЕЛАЕШЬ

Ты создаёшь мир в котором происходит игра. Все 5 уровней — это твоя работа. Ты рисуешь карты: где будут стены, где коридоры, где вода, где шипы. Ты расставляешь предметы: сыр который собирает мышь, кости для собаки, лунку для перехода на следующий уровень. Ты решаешь где стартует мышь, где появляется кот, где бродит собака.

Каждый уровень — это не просто случайный набор стен. У каждого своя атмосфера и своя механика. Первый уровень учит игрока основам. Второй даёт больше пространства. Третий добавляет собаку. Четвёртый полон ловушек. Пятый — финальная битва с боссом.

---

### 🎯 ПОСЛЕДОВАТЕЛЬНОСТЬ ЗАДАЧ

**Задача 1: Создать уровень 1 — Подземелье (обучение)**

Создай файл `levels/level1.js`. Экспортируй из него объект `level1` со всеми настройками.

Карта — это двумерный массив 50 на 50 ячеек. Каждое число означает тип тайла:
- 0 = пусто (можно ходить)
- 1 = стена (нельзя пройти)
- 3 = вода (опасно, можно утонуть)

Нарисуй карту:

Стены по всему периметру — это границы уровня. Внутри — 4 комнаты соединённые дверями (проходами в стенах). Первая комната маленькая и безопасная — здесь стартует мышь. Вторая комната побольше — здесь бродит кот. Третья комната — с несколькими кусочками сыра. Четвёртая комната — самая большая, в ней лунка.

Размеры комнат примерно 8x8 тайлов. Коридоры между ними шириной 2 тайла.

Настройки уровня:
- `name: "Подземелье"` — название
- `hint: "Собери весь сыр чтобы открыть лунку"` — подсказка
- `dogEnabled: false` — собаки нет
- `darknessEnabled: false` — видимость полная
- `trapsEnabled: false` — ловушек нет
- `bossEnabled: false` — босса нет
- `cheeseGoal: 5` — сколько сыра собрать
- `catSpeed: 2.2` — обычная скорость кота
- `visionRadius: 0` — полный обзор

Расстановка предметов:
- `mouseStart: { x: 160, y: 200 }` — мышь в первой комнате
- `catStart: { x: 800, y: 200 }` — кот в третьей комнате
- `cheeses: [...]` — массив из 5 сыров в разных комнатах (координаты x, y)
- `hole: { x: 700, y: 1100 }` — лунка в последней комнате
- `bones: []` — костей нет

**Задача 2: Создать уровень 2 — Улица**

Создай файл `levels/level2.js`. Экспортируй объект `level2`.

Карта 60 на 60 тайлов — больше чем первый уровень. Открытое пространство: меньше стен, больше пустоты. Нарисуй несколько «зданий» — больших квадратов стен по краям карты. Между ними — широкие улицы. В центре — открытая площадь.

Особенность: мало укрытий. Мышь видно издалека. Кот быстрее замечает добычу.

Настройки:
- `name: "Опасные улицы"`
- `hint: "На открытом пространстве сложнее спрятаться"`
- `dogEnabled: false`
- `darknessEnabled: false`
- `trapsEnabled: false`
- `bossEnabled: false`
- `cheeseGoal: 8` — больше сыра чем в первом уровне
- `catSpeed: 2.5` — кот чуть быстрее
- `visionRadius: 0`

Предметы:
- Сыр расставлен широко — по 1-2 в разных углах карты
- Лунка в дальнем углу от старта
- Костей нет

---

**Задача 3: Создать уровень 3 — Дружба**

Создай файл `levels/level3.js`. Экспортируй объект `level3`.

Карта 50 на 50. Круглая арена — в центре большое открытое пространство. По краям — вода (тайлы = 3). Через воду перекинуты мосты (тайлы = 0). В центре арены — остров.

Здесь впервые появляется собака. Она бродит по арене и защищает мышь если её покормить.

Настройки:
- `name: "Друг или враг?"`
- `hint: "Покорми собаку костью — она защитит тебя"`
- `dogEnabled: true` — собака включена
- `darknessEnabled: false`
- `trapsEnabled: false`
- `bossEnabled: false`
- `cheeseGoal: 6`
- `catSpeed: 2.2`
- `visionRadius: 0`

Предметы:
- `dogStart: { x: 400, y: 400 }` — собака на арене
- `bones: [{ x: 200, y: 500 }, { x: 600, y: 300 }, { x: 400, y: 700 }]` — три кости
- Сыр на островах за водой — нужно пройти по мостам
- Лунка на центральном острове

---

**Задача 4: Создать уровень 4 — Ловушки**

Создай файл `levels/level4.js`. Экспортируй объект `level4`.

Карта 45 на 45. Узкие туннели — ширина 1-2 тайла. Много поворотов. По всему уровню расставлены масляные лужи, шипы и исчезающие платформы.

Настройки:
- `name: "Ловушки повсюду"`
- `hint: "Масло ускоряет но делает неуправляемой"`
- `dogEnabled: false`
- `darknessEnabled: false`
- `trapsEnabled: true` — ловушки включены
- `bossEnabled: false`
- `cheeseGoal: 7`
- `catSpeed: 2.3`
- `visionRadius: 0`

Ловушки:
- Масляные лужи: `oilSlicks: [{ x: 300, y: 400, w: 80, h: 40 }, ...]` — 4-5 штук в разных местах
- Шипы: `spikes: [{ x: 500, y: 600 }, ...]` — 6-8 штук в узких проходах
- Платформы: `platforms: [{ x: 200, y: 800, timer: 180 }, ...]` — 3-4 штуки над водой или пропастью

**Задача 5: Создать уровень 5 — Финал**

Создай файл `levels/level5.js`. Экспортируй объект `level5`.

Карта 55 на 55. Большая комната почти на весь экран. В центре — логово босса. По краям — разрушенные стены (неполные, с проходами).

Здесь появляется босс-кот: невидимый, видны только красные глаза. Вокруг него красная аура. Игроку страшно.

Настройки:
- `name: "Последняя битва"`
- `hint: "Красная аура означает что босс рядом"`
- `dogEnabled: true` — собака помогает
- `darknessEnabled: true` — темнота, узкий обзор
- `trapsEnabled: true`
- `bossEnabled: true` — босс включен
- `catInvisible: true` — кот невидим
- `fearAura: true` — красная аура
- `cheeseGoal: 5`
- `catSpeed: 3.0` — босс быстрый
- `visionRadius: 80` — маленький обзор (только 80 пикселей вокруг мыши)
- `bossPhases: 3` — три фазы босса

Предметы:
- Кости для собаки — чтобы было чем кормить защитника
- Сыр спрятан в углах комнаты
- Лунка появляется только после победы над боссом

**Задача 6: Написать функцию получения уровня**

Создай файл `levels/index.js` который собирает все уровни вместе:

- Импортирует level1 из level1.js, level2 из level2.js и так далее
- Создаёт массив `allLevels = [level1, level2, level3, level4, level5]`
- Экспортирует функцию `getLevelConfig(number)` которая возвращает allLevels[number]

### 💬 ПОДСКАЗКИ

Рисуй карту в виде сетки в комментариях кода — так легче ориентироваться. Например:

```
// Карта уровня 1 (50x50)
// 1111111111111111  — стена
// 1000000000000001  — пусто
// 1000000000000001
// 1111111111111111
```

Проверяй проходимость: мысленно пройди от старта до лунки. Все ли коридоры достаточно широкие? Не зажат ли сыр в недоступных местах?

Для тестирования попроси Наставника запустить твой уровень в игре и проверь что мышь может собрать весь сыр и дойти до лунки.
